# Marketing Funnel & Conversion Performance Analysis

## Project Overview
This project analyzes marketing funnel performance using the Bank Marketing Dataset to evaluate customer conversion behavior and identify critical drop-off points in the customer journey.

The objective is to understand how customers move through the marketing funnel and identify opportunities to improve conversion efficiency through data-driven insights.

---

## Problem Statement
Businesses often experience significant customer drop-off during marketing campaigns, reducing overall conversion effectiveness.

This project aims to analyze customer conversion patterns, identify funnel bottlenecks, and generate actionable recommendations to improve campaign performance.

---

## Objectives
- Analyze customer conversion performance
- Measure overall funnel efficiency
- Identify key drop-off points
- Compare communication channel effectiveness
- Evaluate monthly conversion trends
- Analyze campaign frequency impact
- Build an interactive dashboard
- Generate business recommendations

---

## Key Analysis Performed

### 1. Customer Conversion Distribution
Analyzed overall converted vs non-converted customers.

### 2. Conversion by Contact Type
Compared cellular and telephone communication performance.

### 3. Conversion by Month
Identified seasonal conversion trends.

### 4. Campaign Effect Analysis
Evaluated how repeated contact attempts affect conversion.

### 5. Funnel Performance
Measured conversion rate and customer drop-off.
